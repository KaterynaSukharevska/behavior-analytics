# Behavior Analytics MVP

Behavior Analytics MVP is a portfolio project focused on collecting, processing, and visualizing product behavior data.

## Current Status

Initial monorepo foundation is set up. Application implementations are intentionally not added yet.

## Local Development Prerequisites

- Node.js 20+ and npm
- Docker and Docker Compose

## Basic Commands

```bash
npm install
docker compose up -d
docker compose ps
docker compose down
```

## Local Ports

- Dashboard: `3000`
- Demo Site: `3001`
- Ingest API: `4000`
- PostgreSQL: `5432`
