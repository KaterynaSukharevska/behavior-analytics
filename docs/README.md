# Behavior Analytics MVP Documentation

This folder contains product, architecture, technical, and portfolio documentation for the Behavior Analytics MVP project.

Behavior Analytics MVP is a lightweight website behavior analytics product with:

- tracking SDK
- ingest API
- analytics dashboard
- demo website

The project is designed as a real portfolio / career project for demonstrating modern frontend engineering, product thinking, dashboard UI, TypeScript, browser APIs, testing, and privacy-aware product development.
